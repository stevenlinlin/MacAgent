# 早晨启动流程

## 触发条件
- 用户说"早上好"、"早安"、"开始一天"
- 用户明确要求执行早晨流程

## 执行步骤
1. **打开微信**：确保微信已启动并置于前台。
2. **查询天气**：获取指定城市的天气信息。
3. **判断天气**：如果天气晴好（温度>20°C且无雨），则发送一条消息给指定联系人，提醒"今天天气不错，适合出门"；否则提醒"今天天气一般，注意保暖"。
4. **调节音量**：将系统音量调至 30%（避免早晨音量过大）。
5. **返回综合报告**：告知用户所有操作已完成。

## 输出格式
成功：返回一条包含所有操作状态的摘要消息。
失败：返回具体错误信息（如天气API失败、微信未登录等）。
